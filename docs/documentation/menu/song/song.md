# Song menu hierarchy

The Song menu is accessible from `Arranger View` and `Song View` by pressing on the `SELECT ENCODER`

The Song menu contains the following menu hierarchy:

## Master (MASTR)

	- Volume (VOLU)
	- Pan

## Filters (FLTR)

	- LPF
		- Frequency (FREQ)
		- Resonance (RESO)
		- Morph (MORP)
		- Mode (MODE)
			- 12DB Ladder (LA12)
			- 24DB Ladder (LA24)
			- Drive (DRIV)
			- SVF Bandpass (SV_B)
			- SVF Notch (SV_N)
	- HPF
		- Frequency (FREQ)
		- Resonance (RESO)
		- Morph (MORP)
		- Mode (MODE)
			- SVF Bandpass (SV_B)
			- SVF Notch (SV_N)
			- HP Ladder (HP_L)
	- Filter Route (ROUT)
		- HPF2LPF (HPF2)
		- LPF2HPF (LPF2)
		- PARALLEL (PARA)
